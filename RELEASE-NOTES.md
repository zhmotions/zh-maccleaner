# ZH MacCleaner 1.0

First public release. A safe, simple Mac cleaner by ZH Motions.

### Features
- 🧹 **Cleanup** — app caches, logs, browser caches (auto-scans on launch)
- 📦 **Large Files** — find files > 100 MB, send to Trash
- 🗑️ **Uninstaller** — remove an app + all its leftover files
- 👯 **Duplicates** — find identical files, trash the extras
- 🛠 **Maintenance** — free RAM, flush DNS, reindex Spotlight, rebuild Launch DB
- ℹ️ In-app Help & tooltips explaining every tool

### Install
1. Download `ZH-MacCleaner-1.0.dmg` (or the `.zip`) below.
2. Open the DMG → drag **ZH MacCleaner** to Applications.
3. First launch: **right-click → Open** (unsigned — Gatekeeper asks once).

### Notes
- Safe by design: only known cache folders are deleted; your files go to the Trash.
- For full cache cleanup, grant **Full Disk Access** (Settings → Privacy & Security).
- Requires macOS 11+.
