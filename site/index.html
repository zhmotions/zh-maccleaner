Install (takes 20 seconds)
  1. Click Download for macOS above — you get ZH-MacCleaner.pkg.
  2. Right-click the .pkg → Open → Open → Continue → Install. (Right-click the first time so macOS lets it run.) It installs to Applications for you.
  3. First launch only: open Applications, right-click ZH MacCleaner → Open → Open. (After this it opens normally.)
⚠ See "ZH MacCleaner is damaged"?

It's not damaged — macOS just flags new apps from outside the App Store. One-time fix:

  1. Open Terminal (press ⌘ + Space, type Terminal, Enter).
  2. Paste this line and press Enter:
xattr -cr "/Applications/ZH MacCleaner.app"

Then open the app normally. You only do this once.
